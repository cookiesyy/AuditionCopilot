## 在Windows上编译步骤

### 方法1：使用Visual Studio（推荐）

#### 1. 安装Visual Studio
- 下载并安装 Visual Studio 2019 或更高版本
- 安装时选择 ".NET桌面开发" 工作负载
- 确保安装了 .NET Framework 4.8 开发工具

#### 2. 打开项目
1. 解压 `AuditionCopilot-src.tar.gz`（Windows上用7-Zip或WinRAR）
2. 双击打开 `AuditionCopilot.sln` 文件
3. Visual Studio会自动加载项目

#### 3. 还原NuGet包
- 在Visual Studio中，右键点击解决方案
- 选择 "还原NuGet包"
- 等待依赖包下载完成

#### 4. 编译项目
- 点击菜单 "生成" -> "生成解决方案"
- 或按快捷键 `Ctrl+Shift+B`
- 等待编译完成

#### 5. 获取EXE文件
编译成功后，EXE文件位置：
- Debug版本：`AuditionCopilot/bin/Debug/AuditionCopilot.exe`
- Release版本：`AuditionCopilot/bin/Release/AuditionCopilot.exe`

**建议使用Release版本**，性能更好。

### 方法2：使用MSBuild命令行

如果你熟悉命令行，可以直接用MSBuild编译：

```cmd
# 打开"开发人员命令提示符"（Developer Command Prompt）
cd AuditionCopilot
msbuild AuditionCopilot.sln /p:Configuration=Release /p:Platform=x86
```

编译后的EXE在 `AuditionCopilot/bin/Release/AuditionCopilot.exe`

## 运行要求

### 系统要求
- Windows 10 或 Windows 11
- .NET Framework 4.8（Windows 10/11通常已预装）
- 屏幕分辨率：1920×1080

### 依赖文件
运行时需要以下文件（编译后会自动生成）：
- `AuditionCopilot.exe` - 主程序
- `AForge.dll` - 图像处理库
- `AForge.Imaging.dll` - 图像识别库
- `AForge.Math.dll` - 数学计算库
- `DD驱动DLL文件` - 需要单独下载DD驱动

### DD驱动安装
1. 下载DD驱动（搜索"DD驱动下载"）
2. 运行程序后，点击"加载DD驱动"按钮
3. 选择DD驱动的DLL文件

## 使用说明

### 1. 启动游戏
- 打开劲舞团游戏
- 确保游戏窗口标题为 **"劲舞团 华东一区"**
- 如果标题不同，需要修改源码中的窗口名称

### 2. 运行程序
- 双击 `AuditionCopilot.exe`
- 程序会自动查找游戏窗口并定位识别区域

### 3. 加载DD驱动
- 点击"加载DD驱动"按钮
- 选择DD驱动DLL文件
- 等待初始化完成

### 4. 开始使用
- 勾选"自动空格"选项
- 程序会自动识别方向键并按键
- 可以调整空格键按下时机

## 常见问题

### Q1: 编译时提示缺少NuGet包
**解决：** 右键解决方案 -> 还原NuGet包

### Q2: 编译时提示.NET Framework版本不对
**解决：** 安装.NET Framework 4.8 SDK

### Q3: 运行时提示找不到游戏窗口
**解决：**
1. 确认游戏窗口标题是否为"劲舞团 华东一区"
2. 如果不是，修改 `MainForm.cs` 第117行的窗口名称

### Q4: 按键识别不准确
**解决：**
1. 确保游戏分辨率为1920×1080
2. 调整 `MainForm.cs` 第119-120行的坐标偏移量

### Q5: DD驱动加载失败
**解决：**
1. 确保DD驱动DLL文件完整
2. 以管理员身份运行程序
3. 检查杀毒软件是否拦截

## 技术支持

如有问题，请检查：
1. 游戏窗口标题是否正确
2. 屏幕分辨率是否为1920×1080
3. DD驱动是否正确加载
4. .NET Framework 4.8是否已安装
